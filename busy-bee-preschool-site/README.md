# Busy Bee Preschool Center Website

This is a simple static website that can be hosted on GitHub Pages.

## Files

- `index.html` - page content and sections
- `styles.css` - visual design and responsive layout
- `script.js` - mobile menu behavior
- `assets/` - local artwork and icons

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload everything inside this folder to the repository root.
3. In GitHub, open **Settings**.
4. Open **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root`.
7. Save.

GitHub will provide a live website URL after the first deployment finishes.
